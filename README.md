Lingua-Voikko version 0.01
==========================

Perl Interface for Voikko Spell Checker and Lemmatizer.

INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

DEPENDENCIES

This module requires these other modules and libraries:

Voikko
